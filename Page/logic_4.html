GATE NAND

NAND หรือ Not AND จะมี Output เป็น 1 เมื่อ Input ทุกตัวเป็น 0 หรือเป็น 0 เมื่อมี Input ตัวใดตัวหนึ่งเป็น 1 (ผลลัพธ์ที่ได้จะตรงข้ามกับ Gate And)

GATE NOR

NOT หรือ Not OR จะมี Output เป็น 0 เมื่อ Input ทุกตัวเป็น 1 หรือเป็น 1 เมื่อมี Input ตัวใดตัวหนึ่งเป็น 0

GATE XOR

XOR หรือ Exclusive OR จะเป็น Gate OR ที่มีความพิเศษตรงที่ถ้าค่าความจริงเหมือนกัน ผลลัพธ์ที่ได้จะออกมาเป็น 1 เสมอ
