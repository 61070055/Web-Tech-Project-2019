LOGIC

(ตรรกะศาสตร์เชิงคอมพิวเตอร์)

System ?


System คือ ระบบ หรือสิ่งที่กำหนดความสัมพันธ์ของ Input และ Output โดยให้เรามองตัว System เป็นกล่องๆหนึ่ง ซึ่งเราป้อน Input ไปด้านหนึ่ง และกล่องหรือตัวระบบจะทำการประมวลผลและส่งเป็น Output ออกมา

Digital System

ระบบนี้มีลักษณะสำคัญคือ ระดับค่าของ Input และ Output นั้นจะมีจำนวนที่จำกัด และเป็นแบบไม่ต่อเนื่องกัน

Analog System

ระบบนี้มีลักษณะสำคัญคือ ระดับค่าของ Input และ Output จะมีการรับค่าที่ไม่จำกัด และเป็นแบบต่อเนื่องกัน
