Input / variable

การกำหนด Input เป็นการกำหนดว่า Input จะรับค่ามาจากอะไร เช่น Mouse , Keyboard เป็นต้น คำสั่งพื้นฐาน : GET, INPUT
ตัวแปร (Variable) คือการเก็บค่าของชนิดตัวแปรใดๆ ไว้ในหน่วยข้อมูล ประเภทของตัวแปรพื้นฐานก็จะมี
String = ข้อความ
Integer = ตัวเลข
Float / Double = ทศนิยม
Boolean = ค่าความจริง
ซึ่งการกำหนดตัวแปรพื้นฐานของระบบคอมพิวเตอร์ก็จะมีคำสั่งพื้นฐานดังนี้
number = 1, text = "Hello" เป็นต้น
