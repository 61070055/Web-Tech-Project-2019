LOGIC GATE

(ประตูสัญญาณตรรกะ)

LOGIC GATE


ตัวดำเนินการทางตรรกศาสตร์ โดยจะรับ Input อย่างน้อย 1 ตัว มาคำนวณและส่ง Output ออกไป 1 ตัว ถ้าเรานำ Logic Gate มาประกอบกัน จะได้วงจรเรียกว่า Logic Circuit ซึ่งเป็นวงจร Digital รูปแบบหนึ่ง

GATE AND

Output เป็น 1 เมื่อ Input ทุกตัวเป็น 1 หรือเป็น 0 เมื่อมี Input ตัวใดตัวหนึ่งเป็น 0

GATE OR

Output เป็น 0 เมื่อ Input ทุกตัวเป็น 0 หรือเป็น 1 เมื่อมี Input ตัวใดตัวหนึ่งเป็น 1

GATE NOT

Output เป็น 0 เมื่อ Input เป็น 1 หรือเป็น 1 เมื่อ Input เป็น 0
